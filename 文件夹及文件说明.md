# 文件夹及文件说明

本文档详细说明了项目根目录及各子目录下文件夹和文件的作用，旨在帮助开发者快速了解项目结构。

## 根目录

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **backend/** | 文件夹 | 后端项目代码目录（基于 Flask 框架）。 |
| **frontend/** | 文件夹 | 前端项目代码目录（基于 Vue 3 + Vite）。 |
| **docker/** | 文件夹 | Docker 容器化部署相关的配置文件。 |
| **README.md** | 文件 | 项目的说明文档，包含项目介绍、快速开始、技术栈等信息。 |
| **.git/** | 文件夹 | Git 版本控制目录。 |

---

## Backend (后端)
目录：`backend/`

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **app/** | 文件夹 | **核心应用代码目录**，包含所有的业务逻辑模块。 |
| **migrations/** | 文件夹 | 数据库迁移脚本目录（Flask-Migrate/Alembic 生成）。 |
| **scripts/** | 文件夹 | 实用脚本目录，包含如数据库初始化等脚本。 |
| **Dockerfile** | 文件 | 后端镜像构建文件，定义了后端运行环境。 |
| **requirements.txt** | 文件 | Python 依赖包列表。 |
| **gunicorn.conf.py** | 文件 | Gunicorn WSGI 服务器配置文件，用于生产环境启动后端。 |
| **app.log** | 文件 | 应用运行日志文件。 |

### backend/app/
目录：`backend/app/` (核心模块)

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **\_\_init\_\_.py** | 文件 | 这里的 app 工厂函数用于初始化 Flask 应用、注册蓝图和扩展。 |
| **config.py** | 文件 | 应用的配置文件，包含数据库、密钥、Redis、MinIO 等配置。 |
| **extensions.py** | 文件 | 扩展初始化文件（如 SQLAlchemy, JWTManager 等）。 |
| **sync.py** | 文件 | 可能用于数据同步或异步任务的逻辑。 |
| **admin/** | 文件夹 | 管理员相关功能模块。 |
| **auth/** | 文件夹 | 认证与授权模块（登录、注册、Token 管理）。 |
| **users/** | 文件夹 | 用户管理模块。 |
| **ai/** | 文件夹 | AI 助手相关功能模块。 |
| **doc/** | 文件夹 | 文档管理相关模块。 |
| **files/** | 文件夹 | 文件上传与管理模块。 |
| **schedule/** | 文件夹 | 日程与会议管理模块。 |
| **approval/** | 文件夹 | 审批流程模块。 |
| **notice/** | 文件夹 | 公告与消息通知模块。 |
| **tasks/** | 文件夹 | 任务看板/项目管理模块。 |
| **knowledge/** | 文件夹 | 知识库相关模块。 |
| **feedback/** | 文件夹 | 用户反馈相关模块。 |
| **privacy/** | 文件夹 | 隐私空间相关模块。 |
| **utils/** | 文件夹 | 通用工具函数（如时间处理、响应封装等）。 |
| **models/** | 文件夹 | 存放数据库模型定义的目录（部分模型可能分散在各模块中）。 |

## Frontend (前端)
目录：`frontend/`

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **src/** | 文件夹 | **前端源代码目录**。 |
| **public/** | 文件夹 | 静态资源目录（如 favicon 等），不会被 Vite 编译。 |
| **dist/** | 文件夹 | 打包输出目录（构建生产环境代码后生成）。 |
| **Dockerfile** | 文件 | 前端镜像构建文件，通常用于构建 Nginx 镜像托管静态文件。 |
| **vite.config.js** | 文件 | Vite 项目配置文件（插件配置、代理设置等）。 |
| **package.json** | 文件 | 项目元数据及 npm 依赖定义。 |
| **index.html** | 文件 | 单页应用的入口 HTML 文件。 |

### frontend/src/
目录：`frontend/src/`

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **main.js** | 文件 | Vue 应用入口文件，挂载 App 和注册全局插件。 |
| **App.vue** | 文件 | 根组件。 |
| **api/** | 文件夹 | 统一管理 API 请求（Axios 封装及接口定义）。 |
| **views/** | 文件夹 | 页面级组件（路由对应的页面）。 |
| **components/** | 文件夹 | 通用 UI 组件。 |
| **router/** | 文件夹 | Vue Router 路由配置。 |
| **store/** | 文件夹 | Pinia/Vuex 状态管理。 |
| **assets/** | 文件夹 | 静态资源（图片、全局样式文件等）。 |
| **styles/** | 文件夹 | 全局 CSS/SCSS 样式文件。 |

---

## Docker (部署配置)
目录：`docker/`

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| **docker-compose.yml** | 文件 | 容器编排文件，定义了所有服务（Frontend, Backend, MySQL, Redis, MinIO, ES 等）的运行方式。 |
| **nginx.conf** | 文件 | Nginx 配置文件，作为反向代理服务器，处理请求转发（/api -> 后端, / -> 前端）。 |
